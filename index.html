このサイトの使い方

「いま募集中 × 予算が合う × VTuber を描ける」絵師を探す社内ツールです。

並び順「おすすめ順」の基準

「今すぐ頼める × VTuber を描ける」絵師を上に出します。

1主対象（◎）を最優先 ＝「募集中」かつ「イラスト」かつ「VTuber適性が 確実 / 有力」の絵師（カードが金枠）。
2次に受付状況：募集中 → 受付不明 → 休止中。
3同じならVTuber適性スコアの高い順。

ほかに 新着順 / 価格 / 納期 / VTuber適性 / 完了率 / 受注数 / 名前 で並び替えできます。

VTuber適性ランクの意味

  • 確実 名鑑の特定 VTuber 名にヒット（実績が確認できた）
  • 有力 プロフィールに「VTuber / Live2D / 立ち絵 / キャラデザ」等の語
  • 候補 弱い手がかりのみ

※その絵師が描いた可能性の高い VTuber 名は 🔍詳細ビューで確認できます（検索枠に VTuber 名を入れて探すことも可能）。

絞り込み

  • 募集中のみ：いまリクエスト受付中だけ表示（既定）
  • 価格：スライダーで下限〜上限の予算範囲を指定（絵師のイラスト最低額が範囲内なら表示。範囲指定中は価格不明の絵師は出ません）
  • 納期：クリエイター設定の納品期限（リクエスト承認から◯日以内に納品）で絞る。平均実績ではなく本人設定の上限です。データは順次取得中のため、未取得の絵師は絞り込み中は表示されません
  • VTuber適性：「確実のみ / 有力以上 / 候補以上」で絞る
  • 状態：選定ステータス（検討中/打診済/発注済/納品済/見送り）で絞る。⭐絞り込みと併用可。特定の状態を選ぶと受付・価格などの他条件は無視されます（発注済の絵師は受付停止しているのが普通のため）
  • R-18サムネ表示：オフで成人向け作品サムネをぼかす（既定オフ・安全側）。※R-18も受ける絵師も一覧には常に表示されます（SFW作品も描けるため。バッジで判別可）
  • 要目視も表示：価格・受付・ジャンルが不明な絵師を含めるか
  • ⭐自分 / ⭐全員：お気に入りだけに絞る（下記）。絞ると受付・価格 等の他条件は無視して全お気に入りを表示
  • 上の検索枠は名前・@handle・描いた VTuber 名・プロフィール文で絞れます（「Live2D」「立ち絵」等も検索可）

🚫 除外（合わない絵師を弾く）

カード右上の 🚫除外 を押すと、その絵師はこの端末で今すぐ非表示になります。 上に出る「コレクションに反映」のコマンドを巡回PCで実行すると、今後も収集しません。

上部バーの「一覧を見る」（スマホは⚙内の「🚫 除外一覧」）で除外済みを確認できます。 この端末の除外はカード表示され「↩ 解除」でその場で戻せます。コレクション恒久除外（反映済み）は データ本体が削除済みのため @handle の行リスト表示となり、復帰は「復帰コマンド」を巡回PCで実行します。

⭐ お気に入り（3〜4人で共有）

カード右上の ⭐ を押すと、その絵師がチーム全員の共有お気に入りに入り、 誰が付けたか（表示名）も出ます。初回だけ表示名を訊きます（端末に保存）。 上部バーの「表示名」から変更、絞り込みの ⭐自分 / ⭐全員 で候補だけに絞れます。 ⭐は休止中でも隠れません（候補を見失わないため）。「更新」で最新の共有状態に同期します。

➕ 絵師を追加（未掲載の絵師を申請）

X などで見つけた絵師がこのサイトに無いときは、右上の「➕ 絵師を追加」（スマホは⚙内）に Skeb の URL（skeb.jp/@xxxx）を貼ってください。次回の自動更新（毎日11時ごろ）で 収集・掲載され、申請したあなたの⭐に自動で入ります。登録済みの絵師ならその場でカードを表示します。 ※Skeb に存在しない URL や、イラストを提供していない絵師（ボイス専業等）は掲載されません。

🔍 詳細ビュー・選定ステータス・共有メモ

カードの 🔍 ボタンで、その絵師の作品を最大6枚・プロフィール文（先頭200字）・ 選定ステータスとチーム共有メモを開けます。ステータス （検討中 → 打診済 → 発注済 → 納品済 / 見送り）とメモは全員に共有され、 二重打診の防止と「なぜ⭐したか」の記録に使います。カードにも状態チップと 📝 が出ます。

⚠ 発注前に知るべき Skeb の制度

  • リテイク不可：Skeb は「おまかせ」形式で、納品後の修正指示は原則できません
  • 著作権は絵師に帰属：権利譲渡はなく依頼者の利用範囲は限定的です。事務所公式の立ち絵・商用利用が目的の場合は、Skeb 外での直接契約（権利許諾・料金は別相場）が通例です
  • 表示価格は最低額：カードの金額は絵師が設定した「おまかせ最低額」で、キャラデザ・立ち絵の実際の依頼額はその数倍になるのが普通です
  • 上記は 2026-07 時点の要約です。発注前に必ず Skeb 公式の最新規約・利用範囲をご確認ください

データの鮮度

受付状況・価格は収集した時点の情報です。各カードの「確認 ○日前」がその絵師の最終確認日、上部の「データ更新 M/D」が最新ビルド日です。募集は Skeb 側で急に締まることがあるため、発注前に絵師名をクリックして Skeb の最新をご確認ください。

作品サムネ・発注

カードの作品画像はクリックで拡大表示されます（画像は Skeb の参照表示）。拡大内の 「Skebで作品を開く」から作品ページへ。発注は絵師名（または @handle）をクリックして Skeb の正規フローで。社内専用のため URL・パスワードの共有は控えてください。

@handle 横の 𝕏 は Skeb 連携元の X アカウント（同名ハンドル）を開きます。 X 側で改名・移転している場合はリンク切れになるため、その際は Skeb ページから最新のリンクをご確認ください。
