このサイトの使い方

「いま募集中 × 予算が合う × VTuber を描ける」絵師を探す社内ツールです。

並び順「おすすめ順」の基準

「今すぐ頼める × VTuber を描ける」絵師を上に出します。

1主対象（◎）を最優先 ＝「募集中」かつ「イラスト」かつ「VTuber適性が 確実 / 有力」の絵師（カードが金枠）。
2次に受付状況：募集中 → 受付不明 → 休止中。
3同じならVTuber適性スコアの高い順。

ほかに 新着順 / 価格 / VTuber適性 / 完了率 / 受注数 / 名前 で並び替えできます。

VTuber適性ランクの意味

  • 確実 名鑑の特定 VTuber 名にヒット（実績が確認できた）
  • 有力 プロフィールに「VTuber / Live2D / 立ち絵 / キャラデザ」等の語
  • 候補 弱い手がかりのみ

※「○○」の緑タグは、その絵師が描いた可能性の高い VTuber 名です。

絞り込み

  • 募集中のみ：いまリクエスト受付中だけ表示（既定）
  • 価格：イラスト最低額の帯で絞る
  • VTuber適性：上のランクで絞る
  • R-18サムネ表示：オフで成人向け作品サムネをぼかす（既定オフ・安全側）。※R-18も受ける絵師も一覧には常に表示されます（SFW作品も描けるため。バッジで判別可）
  • 要目視も表示：価格・受付・ジャンルが不明な絵師を含めるか
  • ⭐自分 / ⭐全員：お気に入りだけに絞る（下記）。絞ると受付・価格 等の他条件は無視して全お気に入りを表示
  • 上の検索枠は名前・@handle・描いた VTuber 名・プロフィール文で絞れます（「Live2D」「立ち絵」等も検索可）

🚫 除外（合わない絵師を弾く）

カード右上の 🚫除外 を押すと、その絵師はこの端末で今すぐ非表示になります。 上に出る「コレクションに反映」のコマンドを巡回PCで実行すると、今後も収集しません。

⭐ お気に入り（3〜4人で共有）

カード右上の ⭐ を押すと、その絵師がチーム全員の共有お気に入りに入り、 誰が付けたか（表示名）も出ます。初回だけ表示名を訊きます（端末に保存）。 上部バーの「表示名」から変更、絞り込みの ⭐自分 / ⭐全員 で候補だけに絞れます。 ⭐は休止中でも隠れません（候補を見失わないため）。「更新」で最新の共有状態に同期します。

➕ 絵師を追加（未掲載の絵師を申請）

X などで見つけた絵師がこのサイトに無いときは、右上の「➕ 絵師を追加」（スマホは⚙内）に Skeb の URL（skeb.jp/@xxxx）を貼ってください。次回の自動更新（毎日11時ごろ）で 収集・掲載され、申請したあなたの⭐に自動で入ります。登録済みの絵師ならその場でカードを表示します。 ※Skeb に存在しない URL や、イラストを提供していない絵師（ボイス専業等）は掲載されません。

データの鮮度

受付状況・価格は収集した時点の情報です。各カードの「確認 ○日前」がその絵師の最終確認日、上部の「データ更新 M/D」が最新ビルド日です。募集は Skeb 側で急に締まることがあるため、発注前に「Skebで開く」で最新をご確認ください。

作品サムネ・発注

カードの作品画像はクリックで Skeb の作品ページへ（画像は Skeb の参照表示）。 発注は「Skebで開く」から正規フローで。社内専用のため URL・パスワードの共有は控えてください。
